<h1>VidJot</h1>

Node.js & Express App for creating video ideas using Node.js, Express, Mongoose and some other packages.

<h2>Version: 1.0.0</h2>

<h3> Usage</h3>

$ npm install

$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000

</h3>MongoDB</h3>

Open "config/keys.js" and add your MongoDB URI, local or Atlas
